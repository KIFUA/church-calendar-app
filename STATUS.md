# Статус проекту: Церковний Календар

## Що вже зроблено:
- Створено інтерфейс календаря (день, тиждень, місяць).
- Реалізовано систему груп подій, музичних груп та служителів.
- Додано режим адміністратора з паролем (пароль береться з `VITE_INITIAL_AUTH_TOKEN`).
- Інтегровано Firebase Firestore для збереження подій та налаштувань.

## Поточна проблема:
- Дані не зберігаються після перезапуску сторінки. 
- Плутанина між Realtime Database та Firestore. Додаток використовує **Cloud Firestore**.

## Наступні кроки:
1. Перейти в розділ **Firestore** у меню Firebase (не Realtime Database).
2. Створити базу даних Firestore (якщо ще не створена).
3. Налаштувати правила (Rules) саме у розділі Firestore.
